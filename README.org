* SRFI 90: Extensible hash table constructor

** by Marc Feeley

This repository hosts [[http://srfi.schemers.org/srfi-90/][SRFI 90]]: Extensible hash table constructor, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /final/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-90/srfi-90.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-90/][join the SRFI-90 mailing list]] and send your message there.
