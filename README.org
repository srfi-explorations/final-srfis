* SRFI 71: Extended LET-syntax for multiple values

** by Sebastian Egner

This repository hosts [[http://srfi.schemers.org/srfi-71/][SRFI 71]]: Extended LET-syntax for multiple values, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /final/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-71/srfi-71.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-71/][join the SRFI-71 mailing list]] and send your message there.
