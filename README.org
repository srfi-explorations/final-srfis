This repository hosts [[http://srfi.schemers.org/srfi-63/][SRFI 63]]: Homogeneous and Heterogeneous Arrays, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-63/srfi-63.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-63/][join the SRFI-63 mailing list]] and send your message there.
